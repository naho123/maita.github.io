¿Qué es una Dirección IP y cómo funciona?

Lo Protocolos de Internet (Internet Protocol) son las anclas de internet, conectan a la red mundial y apuntalan toda su vida digital. De hecho, es imposible conectarse a internet sin una dirección IP.

Análisis de las Direccines IP

Suelen consistir en 4 números del 0 al 255, separados por puntos. Dentro de una Dirección IP, puede ver el ID de red, asignado a su red y el ID del Host, el identificador único a cada dispositivo conectado a esa red específica.
Este es un ejemplo de dirección IP: 192.168.1.3

ID de Red

Es la parte de una dirección IP con la que se identifica la red que se esta utilizando para conectarse a internet, es decir los 3 primeros números.

ID del Host(afitrión)

Una red puede ser tan pequeña como dos ordenadores conectados entre sí o tan grande como internet en si misma, la ID del anfitrión indica el dispositivo concreto que esta uando para conectarse a una red.

Ejemplo:

192.168.1.3

Los números de color azul representan al ID de la red y los de color rojo representan el ID del Host.
